# MailWhere v0.4.0

## Highlights

- Tray `열기` now opens the unified 업무 보드 instead of a separate shell/tab surface.
- Primary filters are `오늘`, `이번 주`, `날짜 없음`, `전체`.
- Task rows are compact: title, human due label, sender, and right-aligned `열기` / `나중에` / `보관`.
- Review candidates, settings, and developer tools open as separate windows.
- Settings now use product copy (`AI 분석`, `주소`, `AI 서버 인증`) and support direct API key or environment-variable auth.
- Edit/date/manual dialogs show `저장(Enter)` and `취소(Esc)`.
